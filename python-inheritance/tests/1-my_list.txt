The ``1-my_list`` module
======================

Using ``MyList``:

    >>> MyList = __import__("1-my_list").MyList
    
    >>> # 1. Check Instantiation & List Inheritance
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

    >>> # 2. Check __str__ format
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> print(my_list)
    [1, 4, 2]

    >>> # 3. Check print_sorted with no sorted append with negative number
    >>> my_list.append(-5)
    >>> my_list.print_sorted()
    [-5, 1, 2, 4]

    >>> # 4. Check print_sorted with empty list
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

    >>> # 5. Check print_sorted returns a new list (doesn't modify original)
    >>> print(my_list)
    [1, 4, 2, -5]
